# LeetCode Solutions

A personal repository for storing my LeetCode problem solutions and practice.

## Structure

Solutions are organized by difficulty level:

```
├── easy/         # Easy difficulty problems
├── medium/       # Medium difficulty problems
└── hard/         # Hard difficulty problems
```

## Format

Each solution file includes:
- Problem number and title
- Problem description or link
- Solution code
- Time and space complexity analysis

## Languages

Solutions are primarily written in Python.

## Usage

Each solution can be run independently. Navigate to the specific problem file and execute it with Python:

```bash
python3 <problem_file>.py
```
